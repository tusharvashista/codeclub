Resources

Scratch Resources

Scratch provides a library of sound effects, sprites, and backgrounds that you can use in your project. To access these resources, click on the "Choose a Sprite" or "Choose a Sound" buttons in the editor and select "Library."

OpenGameArt

OpenGameArt is a community-driven website that provides free, open-source game assets. You can find sound effects, sprites, and backgrounds that you can use in your project. All assets are licensed under Creative Commons or public domain.

Freesound

Freesound is a database of royalty-free sound effects that you can use in your project. You can search for sound effects by keyword, license, and duration. All sounds are licensed under Creative Commons.

Kenney

Kenney is a website that provides free game assets, including sprites, backgrounds, and sound effects. All assets are licensed under Creative Commons Zero (CC0), which means that you can use them for any purpose without attribution.

GameDev Market

GameDev Market is a marketplace for game assets. However, they also offer a free section that includes game assets, music, and sound effects that you can use in your project.

GitHub

GitHub is a code-sharing platform that allows you to find and share game development code. You can find sample codes for Scratch game development projects on GitHub.

ScratchEd

ScratchEd is a community of educators who teach with Scratch. They provide resources, such as lesson plans and tutorials, that can help you with your Scratch game development project.
